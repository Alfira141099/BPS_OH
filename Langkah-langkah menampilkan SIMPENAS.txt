1. install xampp
2. pindahkan folder "BPS_OH" ke dalam folder "xampp-htdocs"
3. buka xampp, klik start pada Apache dan MySQL
4. database menggunakan phpmyadmin, import database dari foleder "BPS_OH"
5. file database ada di dalam folder "BPS_OH-database" dengan nama bps_oh.sql
4. Buka browser, ketik "localhost/BPS_OH" maka akan muncul tampilan SI nya
